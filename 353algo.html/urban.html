Objective

To enhance accessibility, reduce travel costs, and prioritize green space projects for sustainable development.

Steps and Applications

1. Optimizing Green Space Connectivity

Using Kruskal’s Algorithm, planners developed a Minimum Spanning Tree that connects green spaces with minimal road construction costs. View Code

2. Analyzing Accessibility

The Floyd-Warshall Algorithm identified the shortest paths between key locations. View Code

3. Prioritizing Development Projects

Projects were ranked using Merge Sort for timely execution. View Code

Impact

  • Reduced costs of developing green spaces by 20% using MST.
  • Improved accessibility, reducing travel times by 15%.
  • Timely execution of high-impact projects.
