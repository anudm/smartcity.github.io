Business Case: Agriculture Case Study

Exploring the Role of Agriculture in Sustainable Urban Development

Overview

Sub-task: Agricultural

SDG Goal: 11

Targets: 11.3, 11.A

Indicators: 11.3.1, 11.A.1

Agriculture in cities and nearby areas supports SDG 11 by creating green spaces, reducing food insecurity, and improving the environment. Below are the specific targets and indicators:

  • Target 11.3: Better urban planning integrates urban agriculture to optimize land use.
  • Indicator 11.3.1: Land consumption vs. population growth.
  • Target 11.A: Agriculture connects urban areas with rural regions for sustainable development.
  • Indicator 11.A.1: Cities with regional plans including agriculture.

Algorithm Usage

An optimal route is prepared for the agricultural officer using Dijkstra's Algorithm. The code is implemented using a Heap data structure, achieving an efficiency of |E|log|V|. This algorithm demonstrates how real-world logistics can benefit from efficient planning.

To search agricultural offices and farms, the BFS algorithm is used. The worst case involves O(nm) complexity. You can view the code for Dijkstra's algorithm HERE and the BFS algorithm HERE.

Alternative algorithms like KMP and Boyer-Moore are also applicable for specific scenarios.
